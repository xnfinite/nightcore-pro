# Template Seller Resource Map

Use this map when the product is a spreadsheet, planner, design template, worksheet, calculator, prompt pack, or reusable creator resource.

## Quick Answer

Template sellers should choose the selling path by buyer use, delivery clarity, update needs, marketplace fit, and support boundary. A simple checkout can be enough, but marketplace discovery or a storefront can fit when the catalog, audience, or update workflow needs it.

## Use This When

- You have a template or reusable resource but the selling path is unclear.
- You are choosing between marketplace discovery and owned checkout.
- You need to define delivery, update notes, license boundaries, or support expectations.
